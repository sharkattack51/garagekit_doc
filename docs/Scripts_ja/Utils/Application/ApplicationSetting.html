    ApplicationSetting

    外部設定ファイル ApplicationSetting.xml を読み込んでアプリケーションに反映できます。xml は MonoBehaviour の Awake() で読み込まれるため、Start() 以降のタイミングで利用することができます。

    namespace GarageKit
public class ApplicationSetting : MonoBehaviour
